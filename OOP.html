Object-Oriented Programming

Object-Oriented Programming is a type of computer-programming modelled on classes and objects. With Object-Oriented Programming, we can create classes (blueprints) and instances of classes (objects). These objects have certain variables (attributes) and behaviours (methods) that are written in their code. To create an object, we must use a constructor that is already written in the class. On top of this, classes can inherit data from a super class in the same way that a chihuahua inherits attributes and behaviours of a dog such as fur colour and barking.


Key Facts

  • Classes often represents things that are nouns (i.e Animals, Vehicles and Countries)
  • Methods often represent things that are verbs (i.e. barking, travelling and growing)
  • Objects are stored into RAM and the address of that storage is stored into a reference variable.
