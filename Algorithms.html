Algorithms

An algorithm is commonly known as a process to be followed to solve problems. In Java, an algorithm is the same thing, steps or instructions you should follow to solve a problem. In Java, we have many different types of Algorithms including:

  1. Swap Algorithms
    • Allows the end user to switch values in two, separate variables using a temporary variable.
  2. Copy Algorithms
    • Can be used to create a duplicate version of a Data Structure.
  3. Search Algorithms
    • Can search a data structure until a target value is found.
  4. Sorting Algorithms
    • Can sort a data structure in order as long as the data type is comparable(e.g. Integers, Doubles or Strings)
  5. Accumulating Algorithms
    • Can find the sum of the values within a data structure.

Key Facts

  • Algorithms and their effectiveness can often be compared using time complexity and space complexity.
    • Time complexity determines the speed at which an algorithm will run.
    • The fastest time complexity would be O(1) which requires the same amount of time regardless of the input size
    • One of the slowest time complexities is O(2^n) which as the input size increases, the time to process the input grows exponentially.
  • There are infinite amounts of algorithms.

    Definition: An algorithm is a defined process on how to solve a problem. This algorithm can include calculation, data processing or modification and other variety of tasks. In the above example, we will be discussing specifically the Bubble Sort Algorithm

    Analysis: Bubble Sort is one of the simplest sorting algorithms developers can use to sort Data Structures. Bubble Sort sorts its elements by comparing the ith index with the i+1 index. If one is larger than the other, it swaps them. It has a Worst Case Time Complexity(When the array is reverse sorted) of O(n^2) meaning the more input this algorithm needs to sort through, the longer it will take. It has a Best Case Time Complexity of O(n), when the data structure is already sorted.

    Explanation: In the example listed above, we have an array of Students. We are looking to sort these Students by name, alphabetically. Because we are dealing with a small amount of Students, bubble sort will be perfect for this problem.
