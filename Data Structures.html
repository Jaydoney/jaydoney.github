Data Structures

In Java, like variables, Data Structures can be used to store information. Unlike variables, rather than storing a single value of a dataType, Data Structures such as Arrays and ArrayLists can store a list of DataTypes.


Key Facts

  • In the AP Course, we deal with three types of Data Structures, the Array, which stores a row of data, the 2-D Array which stores a grid of data(Array within an Array) and the ArrayList which is resizeable.
  • Arrays can be traversed using the for-loop.
  • You can find the length of an array using arrayName.length to access the field that stores the array length.
  • Using an index not within the range of any Array will throw an ArrayIndexOutOfBoundsException.
  • ArrayLists can only hold objects(If you want an ArrayList to hold Integers or Doubles, they must be created using their class constructors to instatiate them in the ArrayList)
  • Once Array lengths are set, they cannot be changed however Array Lists are resizable.
